<h1 align="center">LMS - Forum</h1>

## Description

  <p>Learning Management System Forum - full stack web application for managing university related activities with an integrated forum</p>
  </p><br>
  
## Prerequisites
- Install [Docker](https://www.docker.com/)

## Running the app

```
docker-compose up
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
