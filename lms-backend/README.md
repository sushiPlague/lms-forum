<h1 align="center">LMS - Forum</h1>

## Description

  <p>Learning Management System Forum - full stack web application for managing university related activities with an integrated forum</p>
  </p><br>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
